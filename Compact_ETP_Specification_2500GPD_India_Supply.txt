4. SYSTEM DESIGN — TREATMENT TRAIN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

UNIT 1 — INLET SCREENING CHAMBER
----------------------------------
Type                : Manual Bar Screen + Basket Strainer
Screen Opening      : 6 mm bar screen + 2 mm basket
Material            : SS 304 (for USA food plant environment)
Purpose             : Remove food solids, packaging debris, rags
Dimensions          : 600 mm W x 800 mm L x 900 mm D
Mounting            : In-ground concrete chamber (site civil)
Cleaning            : Manual, daily
India Supply Note   : SS 304 fabricated channel with removable screen

UNIT 2 — UNDERGROUND GREASE TRAP
----------------------------------
Type                : Passive Gravity Grease Interceptor
Capacity            : 1,500 gallons (5,680 litres)
                      (Sized for 1-day peak FOG load + safety factor)
Material            : HDPE or FRP (corrosion-resistant, lightweight for shipping)
Dimensions (approx) : 2,400 mm L x 1,200 mm W x 1,500 mm D
Inlet/Outlet        : 4" ANSI flanged (to match US plumbing standards)
Baffles             : 2 internal baffles — SS 304
Venting             : 2" vent pipe connection (site-provided)
Access              : 2 x 24" manhole covers (HDPE)
FOG Removal         : 85–90%
Pumping Frequency   : Once per week by licensed IL grease hauler
Certifications      : PDI G-101 standard compliance recommended
India Supply Note   : FRP tank preferred for export (lighter, cheaper shipping)
                      HDPE also acceptable. Avoid concrete (too heavy for export)

UNIT 3 — EQUALIZATION + pH NEUTRALIZATION TANK
------------------------------------------------
Capacity            : 7,500 litres (2,000 gallons) — ~1 day storage
                      Provides flow & load equalization
Type                : Above-ground skid-mounted tank
Material            : FRP or PE (UV stabilized)
Shape               : Vertical cylindrical with conical bottom
Dimensions (approx) : Dia 1,800 mm x Height 3,000 mm
Mixing              : 1 x 0.75 kW submersible mixer (SS 304 shaft)
pH Control System   :
  - Online pH sensor: 2 x pH probe (Hach/equivalent) with transmitter
  - Acid dosing pump: 0.5 LPH diaphragm metering pump (HCl or Citric acid)
  - Alkali dosing pump: 0.5 LPH diaphragm metering pump (NaOH)
  - Dosing tanks: 2 x 100L HDPE tanks with level indicators
  - pH controller: PLC-based or standalone pH controller
Level Control       : Float switch (high/low) + ultrasonic level sensor
Transfer Pump       : 1 x 0.75 kW SS centrifugal pump (duty) +
                      1 x 0.75 kW (standby) — ANSI/NPT flanges
India Supply Note   : Tank from Sintex / Supreme / Vectus or custom FRP.
                      Pumps from Kirloskar/Grundfos India.
                      pH probes: Hach India or equivalent.

UNIT 4 — MICRO DAF UNIT (DISSOLVED AIR FLOTATION)
---------------------------------------------------
Design Flow         : 5–7 GPM (19–26 LPM) continuous
Type                : Compact skid-mounted micro-DAF
Vessel Material     : SS 304 (inner surfaces) / MS with epoxy (outer)
Vessel Dimensions   : 1,200 mm L x 800 mm W x 1,500 mm H
Operating Pressure  : 4–6 bar (recycle pressure)
Recycle Ratio       : 30–50% of treated flow
Air Dissolving Tube : Stainless steel, 4 bar rated
FOG Removal         : 85–90%
TSS Removal         : 70–80%
BOD Reduction       : 35–45%
Chemical Dosing     :
  - Coagulant: Alum (Aluminium Sulphate) — 50–200 mg/L dose
    Dosing pump: 1 LPH diaphragm type
    Dosing tank: 100L HDPE
  - Polymer (flocculant): Polyacrylamide — 2–5 mg/L dose
    Dosing pump: 0.5 LPH diaphragm type
    Dosing tank: 50L HDPE
Float Sludge        : Collected in 500L sludge holding tank (HDPE)
                      Pumped out 2x/week to licensed waste hauler
Effluent Outlet     : 2" NPT (US standard) flanged connection
Electrical          : 120/240V 60Hz (US standard) — specify at order
Motor               : 0.75 kW recycle pump + air compressor (0.37 kW)
Skid                : MS epoxy-painted skid, 1,500 x 1,000 mm footprint
India Supply Note   : Several Indian vendors make compact DAF:
                      - Thermax (Pune)
                      - Ion Exchange (Mumbai)
                      - WPL India / Enviro Concept
                      - Custom fabrication by any SS fabricator in Rajkot/Ahmedabad
                      KEY: Must specify 120V/60Hz motors for USA
                      Or supply bare skid + Indian motors, rewire in USA

UNIT 5 — AEROBIC BIOTREATMENT (PACKAGED SBR / CONTACT AERATION)
-----------------------------------------------------------------
Type                : Packaged Contact Aeration + Settling
                      (Simpler than SBR for this scale; suitable for India export)
Design Volume       : 7,500 litres (aeration) + 2,500 litres (clarifier)
HRT (Aeration)      : ~18–24 hours
Tank Material       : FRP (GRP) or HDPE
Aeration System     : Fine bubble diffusers (EPDM membrane disc)
Blower              : 1 x 0.75 kW regenerative blower (oil-free)
                      + 1 x 0.75 kW standby
DO Target           : 2–3 mg/L in aeration zone
MLSS Target         : 2,000–3,000 mg/L
BOD Removal         : 85–92%
COD Removal         : 80–88%
Sludge Return       : Air-lift return from clarifier to aeration
                      (No pump required — simple air-lift design)
Clarifier           : Integrated tube settler / inclined plate settler
                      SS 304 tube modules
Effluent Quality    : BOD < 50 mg/L, TSS < 50 mg/L (post-aeration)
Sludge Wasting      : Weekly pump-out to sludge holding tank
India Supply Note   : Biotreatment tank from FRP fabricator (Ahmedabad/Rajkot)
                      Blower from Everest Blowers / Robuschi India
                      Diffusers from SSI Aeration / Enviro Synergies India

UNIT 6 — EFFLUENT MONITORING + DISCHARGE UNIT
-----------------------------------------------
Type                : Online monitoring panel + sample port
Parameters Monitored: pH, Flow (totalizer), Temperature
pH Sensor           : Submersible probe with 4–20 mA output
Flow Meter          : Magnetic flow meter (2") — US NPT flanged
                      (Required by most IL POTWs for permit compliance)
Totalizer           : Electronic, with pulse output for data logging
Sample Port         : 2" ball valve with sample cock (for grab sampling)
Discharge Connection: 4" ANSI flanged (to connect to building drain/sewer)
Panel               : NEMA 4X enclosure (IP66) — for US indoor/outdoor use
                      120V/60Hz power (USA standard)
India Supply Note   : Flow meter from Krohne India / Endress+Hauser India
                      Panel: Fabricate in India; specify NEMA 4X + UL-listed components
